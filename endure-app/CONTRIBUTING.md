# Contributing to ENDURE

Thank you for helping make ENDURE better! Here's how to contribute.

## 🏷️ Nominating a Brand

Open an **Issue** with the title: `[Brand Nomination] Brand Name`

Include:
- Brand name and website
- Category (Outerwear, Denim, Footwear, etc.)
- Evidence for each of the 7 criteria
- Suggested durability score with rationale
- Cost-Per-Wear estimate

## 🐛 Reporting an Error

If a brand score, material spec, or link is wrong:
- Open an Issue with title: `[Data Error] Brand Name`
- Describe what's wrong and provide a source

## 🔧 Code Contributions

1. Fork the repo
2. Make your changes in `index.html`
3. Test on both desktop and mobile (Chrome DevTools mobile view)
4. Submit a Pull Request with a clear description

## ✅ Brand Acceptance Checklist

- [ ] Meets all 7 criteria pillars
- [ ] Score has a documented breakdown
- [ ] All material claims are sourceable
- [ ] URL links to official brand website
- [ ] Tested on iOS Safari and Android Chrome
